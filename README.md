# Carbon

## Description
An experiment to create a usable emacs-inspired editor with Clojure

## Running
*ONLY MacOS is supported right now *
#### Dependencies
* Clojure
* Make

#### Running
```bash
# If you are on M1
make macos-arm

# If you are on Intel
make macos
```

As no save / open functionality is supported this is the way to start right now.
